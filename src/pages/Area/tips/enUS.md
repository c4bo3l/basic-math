# Equations

## Square
### Equation

```Katex
A = r * r
```

Where *r* is the side length

## Rectangle
### Equation

```Katex
A = w * l
```

Where *w* is the width and *l* is the length.

## Circle
### Equation
```Katex
A = \pi * r * r
```

Where *r* is the radius.

```Katex 
\pi = 3.14
```

or

```Katex
\pi = \frac{22}{7}
```

## Triangle
### Equation

```Katex
A = \frac{1}{2} * b * h
```
Where *b* is the base length and *h* is the height.
