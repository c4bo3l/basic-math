# Rumus-Rumus

## Kotak
### Rumus

```Katex
L = r * r
```

Dimana *r* adalah panjang rusuk.

## Persegi Panjang
### Rumus

```Katex
L = w * l
```

Dimana *w* adalah lebar dan *l* adalah panjang.

## Lingkaran
### Rumus

```Katex
L = \pi * r * r
```

Dimana *r* adalah radius lingkaran.
```Katex 
\pi = 3.14
```

atau

```Katex
\pi = \frac{22}{7}
```

## Segitiga
### Rumus

```Katex
L = \frac{1}{2} * a * t
```

Dimana *a* adalah panjang alas dan *t* adalah tinggi
