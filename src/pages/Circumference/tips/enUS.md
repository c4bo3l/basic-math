# Equations

## Square
### Equation

```Katex
C = 4*r
```

Where *r* is the side length

## Rectangle
### Equation

```Katex
C = 2 * (w+l)
```

Where *w* is the width and *l* is the length.

## Circle
### Equation
```Katex
C = 2 * \pi * r
```

Where *r* is the radius.

```Katex 
\pi = 3.14
```

or

```Katex
\pi = \frac{22}{7}
```

## Triangle
### Equation

```Katex
C = a + b + c
```
