# Rumus-Rumus

## Kotak
### Rumus

```Katex
K = 4*r
```

Dimana *r* adalah panjang rusuk.

## Persegi Panjang
### Rumus

```Katex
K = 2 * (w+l)
```

Dimana *w* adalah lebar dan *l* adalah panjang.

## Lingkaran
### Rumus

```Katex
K = 2 * \pi * r
```

Dimana *r* adalah radius lingkaran.
```Katex 
\pi = 3.14
```

atau

```Katex
\pi = \frac{22}{7}
```
