# Concept

Multiplication is basically the sum of a number.

## Explanation
```
3 x 5 means
The sum of 3 fives -> 3 + 3 + 3 + 3 + 3
or
The sum of 5 threes -> 5 + 5 + 5 
```

# Multiplication 5

> [a]5 x [a]5 = (([a] + 1) x [a])25

## Examples

### Example #1
```
25 x 25
a = 2
((2 + 1) x 2) = 3 x 2 = 6
Result: 625
```

### Example #2
```
35 x 35
a = 3
((3 + 1) x 3) = 4 x 3 = 12
Result: 1225
```

### Example #3
```
75 x 75
a = 7
((7 + 1) x 7) = 8 x 7 = 56
Result: 5625
```

# Multiplication By 11

> [a][b] x 11 = [a]([a] + [b])[b]

## Examples
### Example #1
```
11 x 11
a = 1
b = 1
a + b = 2
Result: 121
```
### Example #2
```
35 x 11
a = 3
b = 5
a + b = 8
Result: 385
```

### Example #3
When the sum of the number greater than 9, the excess number goes to the front number

```
78 x 11
a = 7
b = 8
a + b = 15
Result: (7 + 1)58 = 858
```
